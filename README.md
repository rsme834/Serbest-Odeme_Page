# Modern Payment System 💳

A modern, responsive payment link generator with cart management.

## Features ✨
- 👤 Payer information collection
- 🛒 Dynamic cart management
- 💰 Payment link generation
- 📱 Responsive design
- 🎨 Beautiful animations

## Tech Stack 🛠️
- React.js
- CSS Modules
- React Router
- React Hot Toast


## Installation 🚀

\`\`\`bash
npm install
npm start
\`\`\`
